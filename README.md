# Matthew Kwan - Personal Website

This is the source code for my personal website, hosted at [matthew-kwan.github.io](https://matthew-kwan.github.io).

## Overview

This website serves as my digital portfolio and resume, showcasing my projects, skills, and professional experience. It is designed to be clean, responsive, and user-friendly.

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: For styling and layout, using Flexbox and Grid.
- **Google Fonts**: Typography (Inter).

## Usage

To view the website locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/Matthew-Kwan/Matthew-Kwan.github.io.git
    ```
2.  Open `index.html` in your web browser.
